# Upwork Tool

A Claude AI-powered tool for freelancers to evaluate job fit, write proposals fast, and track connects — built as a single HTML file, no install required.

## What it does

**Job evaluator** — paste any Upwork job post and get a fit score (0-100) with skill, rate, and scope breakdowns, matched and missing skills, red flags, and a go/no-go recommendation.

**Proposal writer** — generates a tailored proposal in your voice based on your profile and the job post. Choose from Direct, Warm, or Senior tone.

**Connects tracker** — log applications, track your balance, and monitor your response rate over time.

## How to use it

1. Download `upwork-tool.html`
2. Open it in any browser
3. Get a free Anthropic API key at [console.anthropic.com](https://console.anthropic.com/settings/keys)
4. Paste your key into the setup field — it saves locally and only needs to be done once

## Requirements

- A free [Anthropic API key](https://console.anthropic.com/settings/keys)
- Any modern browser
- No install, no backend, no dependencies
